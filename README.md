# VetTech Opportunity Radar

VetTech Opportunity Radar is a Python-based data project focused on veterans transitioning into the technology field. The project uses the Adzuna API to collect remote technology job listings over a two-week period and analyze which roles, skills, salary ranges, companies, and job categories appear the most for support-focused and entry-level technology positions.

The project will first be built and tested locally in VS Code using Python, Requests, Pandas, and Matplotlib or Streamlit. After the local version works, the project will expand into AWS by using Lambda for automation, EventBridge for daily scheduling, DynamoDB for cleaned job records, and S3 for raw data backups.

The main goal is to better understand what the remote tech job market is asking for while building practical experience with APIs, automation, cloud tools, data collection, analysis, visualization, and GitHub version control.
